# VeriFire — Propuesta de Tesis

Plataforma SaaS multi-backend (Edge / Cloud / Kubernetes) de inferencia verificable con zkML para redes DePIN de monitoreo ambiental.

**Caso de uso ancla:** detección temprana de incendios en las Sierras de Córdoba.

## 📄 Propuesta

🌐 **Sitio publicado:** [https://dpetrocelli.github.io/verifire/](https://dpetrocelli.github.io/verifire/)

El sitio tiene un toggle entre dos vistas:

- **Resumen** (~5 min de lectura) — info core para mandar rápido.
- **Completa** — propuesta académica completa con diagramas (Venn de campos, arquitectura de servicios, frente de Pareto, Gantt), smart contract Solidity, anexos y plan de difusión open-source.
